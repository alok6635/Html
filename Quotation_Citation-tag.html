Quotation and Citation Elements -

  1. blockquote
  2. q
  3. abbr
  4. address
  5. cite
  6. bdo

blockquote -

blockquote element defines a section that is quoted from another source.

q -

q tag defines a short quotation.

abbr -

abbr tag defines an abbreviation or an acronym, like "HTML", "CSS", "Mr.", "Dr.", "ATM".

address -

address tag defines the contact information for the author/owner of a document.

cite -

cite tag defines the title of a creative work (Ex- a book, a poem, a movie, a painting.A person's name is not the title of a work.

Browsers usually display cite elements in italic.

bdo -

bdo stands for Bi-Directional Override.bdo tag is used to override the current text direction

HTML -

HTML is a markup language it is used to create a webpages to present the content on the World Wide Web.Invented by Tim Berners Lee. html documents are made up two things: content and tags..

DOCTYPE declaration -

defines that this document is an html5 document. it's help browsers to how display web page correctly.

Head -

head element contains meta information about the HTML page. head element is a container for the title,style,meta,link,script,base etc.

Meta -

specify the character set,page description,keywords,viewport settings. and used for search engine optimization to tell the search engine about the page contents.

Title-

title specifies a title for the HTML page. title is important for (SEO).its decide the order when listing pages in search results.

Body -

body defines the document's body, and is a container for all the visible contents, such as headings, paragraphs, images, hyperlinks, tables, lists, etc.

Tags -

Tags are the primary component of the HTML that defines how the content will be structured.these are two types open & close tags.

Element -

HTML elements are defined by a starting tag,some content and a closing tag.

Container tags -

Container tags is opening tag some content and a closing tag.

Empty tags or void elements -

Empty tags are the tags that do not have any content and closing tag.ex-br,hr,img,base,meta,input.

Hr tag -

defines a thematic break in an HTML page, and used to separate content in a page.

Br tag -

br element defines a line break.br if you want a line break (a new line) without starting a new paragraph.

Attributes -

attributes provide additional information about html elements.

src, alt, width, height is common attribute of the image tag.

src attribute specifies the path to the image.

two ways to specify the url in the src attribute-

Absolute url - Links to an external image that is hosted on another website.

Relative url - Links to an image that is hosted within the website.

specifies an alternate text for an image.

img tag should also contain the width and height attributes.

href (hypertext reference) is the common attribute of a tag.

a tag defines a hyperlink.href specifies the url of the page the link goes to.

style Attribute -

style attribute is used to add styles to an element, such as color, font, size.

lang Attribute -

lang attribute inside the tag, to declare the language of the Web page.

title Attribute -

defines some extra information about an element.

Difference between link tag and a tag -

link defines a link between a document and an external resource.

three ways- Active Link – red and underlined. Visited Link – purple and underlined. Unvisited Link – blue and underlined.

a tag is used to create hyperlinks. a tag create a path between two sections or two different html web pages.

To open an url into a new tab in the browser upon a click,we need to add target attribute.

a tag have target attribute. target attribute have four values-

1. blank: opens in a new window or tab.

2. self: default.

3. parent:opens in a parent frame.

4. top:opens in a full-body window.

Block-level Elements-

A block-level element always starts on a new line and takes up the full width available.

Ex -

article,aside,blockquote,canvas,fieldset,figcaption,figure,pre,video.para, h1 to h6

Inline Elements -

An inline element does not start on a new line.its only takes up necessary width.

Ex -

abbr,b,bdo,br,button,cite,code,dfn,em,img,input,label, q,script,select,small,span,strong,sub,sup,textarea.

Note - every element have a default value.

Difference b/w div and span -

div element is block elements.span element is an inline element. span,div has no required attributes, but style,class and id are common.these tags are not semantic tags.

Difference b/w session and local storage -

session storage is available only when a browsers tab is open.ex-use login details,tokens etc.

local storage-is available even after closing & opening the browser window. ex-accept cookies etc.

Ans: Display:none-hide the element and remove space. Visibility:hidden- hide the element but it retain space.

Display flex & display grid-

flex-defines in one dimension - either a row or a column.

grid-defines two-dimensional - rows, and columns at the same time.

Difference figure & picture & img tag-

figure tag specifies the self-contained content,like icons,images,code snippets, etc.it is semantic tag. figcaption element is used to add a caption for the figure element.

Picture tag allows you to display different pictures for different devices or screen sizes.

Image tag is used to embed the image in an html documents.

Difference article & section tag -

An article is intended to be independently distributable or reusable.

A section is a thematic grouping of content.

Responsive Web Design -

What is Quirks mode in html5 -

If we do not include the !DOCTYPE element in our html page or document, it will go to quirks mode. In this mode, the html element depends on the browser. Hence the content will be displayed according to the browser

Can we display a web page inside a web page or is nesting of webpages possible

Yes, we can display a web page inside another html web page.iframe tag is used to embed another document within the current html document.

Difference meter & progress -

progress tag should be used when we want to show the completion progress of a task, whereas if we just want a scalar measurement within a known range or fraction value. Also, we can specify multiple extra attributes for meter tags like ‘form’, ‘low’, ‘high’, ‘min’, etc.

Difference UI & UX

User interface is related to look and feel, colors combinations, fonts selections and layout. user cxperience is related to how ui is used, i.e. usability, wire-framing, interaction with ui.

UI focus on coding skills like html, css and java script, but ux focus on interaction with ui.ex- Design of button is ui, but interaction with button is ux.

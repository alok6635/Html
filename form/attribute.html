Method attributes -

method attribute specifies the HTTP method to be used when submitting the form data. form-data can be sent as URL variables (with method="get") or as HTTP post transaction (with method="post")

Note - The length of a URL is limited (2048 characters).GET is good for non-secure data, like query strings in Google.

Ex post method -




Note - POST has no size limitations, and can be used to send large amounts of data.

Target Attribute -

target attribute specifies where to display the response that is received after submitting the form

Target attribute can have one of the following values-

  1. blank
  2. self
  3. parent
  4. top

blank - displayed in a new window or tab.

self - displayed in the current window

parent - displayed in the parent frame

top - displayed in the full body of the window.

Autocomplete Attribute -

autocomplete attribute specifies whether a form should have autocomplete on or off.

When autocomplete is on, the browser automatically complete values based on values that the user has entered before.
